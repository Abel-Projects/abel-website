# Custom Fonts Directory

Upload your TTF font files here, then let me know the file names and I'll configure them for the scrolling text in the Hero component.

## Current Setup:
- Fonts will be loaded via @font-face declarations in `src/index.css`
- Added to Tailwind config as `font-scroll`
- Applied to scrolling background text in Hero component

## Files you can upload:
- Regular weight: `your-font-regular.ttf` or `your-font.ttf`
- Bold weight: `your-font-bold.ttf` or `your-font-700.ttf`
- Other weights/variants as needed

Once uploaded, just tell me the exact file names and I'll handle the rest!
